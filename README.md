# Face Detection and Air Writing System

This project implements real-time face detection with finger-based air writing using OpenCV and MediaPipe.

## Features
- Face detection using Haar Cascade
- Face displayed in a small window
- Air writing using index finger (no mouse)
- Real-time webcam processing

## Technologies Used
- Python 3.10
- OpenCV
- MediaPipe
- NumPy
